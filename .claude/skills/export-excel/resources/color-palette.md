# Color Palette – Excel Export Chuẩn

## Header & Section
| Dùng cho | Hex | Hiển thị |
|----------|-----|---------|
| Column header background | `1F4E79` | Xanh navy đậm |
| Column header font | `FFFFFF` | Trắng |
| Section title row bg | `2E75B6` | Xanh dương vừa |
| Section title font | `FFFFFF` | Trắng |

## Row stripe
| Dùng cho | Hex |
|----------|-----|
| Row chẵn | `DEEAF1` |
| Row lẻ | `FFFFFF` |
| Border màu | `BDD7EE` |

## Phân loại thay đổi tài liệu (BRD/PRD/FRD)
| Loại | Hex | Ý nghĩa |
|------|-----|---------|
| Sửa / Làm rõ / Việt hóa / Di chuyển / Đổi tên | `FFF2CC` | Vàng nhạt |
| Sửa quan trọng / Sửa phân quyền / Mở rộng scope / Làm rõ quan trọng | `FCE4D6` | Cam nhạt |
| Bỏ (xóa nội dung) | `F4CCCC` | Đỏ nhạt |
| Thêm mới / Bổ sung | `D9EAD3` | Xanh lá nhạt |
| Open Question | `F3F3F3` | Xám nhạt |
| Restructure / Thay thế | `EAD1DC` | Hồng nhạt |
| Thông tin / Trung tính | `E8F0FE` | Xanh nhạt |

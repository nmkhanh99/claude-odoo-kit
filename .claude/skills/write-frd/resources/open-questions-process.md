# Open Questions Process

> File này được share bởi write-brd, write-prd, write-frd.
> Xem rule đầy đủ tại: `.agents/rules/doc-open-questions.md`

## Quick Reference — 4 Bước

1. **Hỏi trước** — liệt kê tất cả câu hỏi, chờ trả lời
2. **Phân tích & hỏi tiếp** — nếu câu trả lời tạo ra câu hỏi mới
3. **Chỉ log câu chưa trả lời** — câu đã trả lời → áp dụng vào nội dung
4. **Phân loại Blocking/Non-blocking** — tìm workaround trước khi đánh blocking

## Những thứ HAY bị suy diễn sai — luôn hỏi:
- Tên menu, đường dẫn menu
- Timeout phê duyệt (giờ/ngày)
- Tên người duyệt theo cấp
- Ngưỡng số lượng gây thay đổi quy trình
- Quy tắc đặt mã lô, số phiếu
- Edge cases chưa có trong nguồn
